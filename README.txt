Crowd Threat Signal Processing – EEG & Behavioral Data
This repository contains the EEG and Behavioral data, as well as analysis scripts, for the study investigating the neural temporal dynamics of threat signal processing in crowd faces. 

Independent Variables
Crowd size: crowd face vs. single face

Emotion: angry vs. fearful

Gaze direction: direct gaze vs. averted gaze

Behavioral Data
Participants rated the mean emotional intensity of each face set on a 1–50 scale. The difference score (estimated – actual mean) was calculated as the main dependent variable.

Requirements
EEG preprocessing: MATLAB (R2019b or later) with EEGLAB (v2022.0).
